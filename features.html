CAPABILITY CATALOG / DIFFERENTIATED LAYER

Evidence-driven evaluation and controlled evolution for agent harnesses.

Harness Ranger sits alongside the execution runtime. It makes model and artifact improvement measurable, scoped, reversible, and safe across heterogeneous agent harnesses.

01 / EVALUATE

Measure what changed

Split failures into actionable signals instead of one opaque score.

Multi-stage evaluation triageSeparates context, extraction, evidence selection, answer, judge disagreement, and safety regressions.
Cross-judge supportCaptures agreement, disagreement, and cross-model-family bias without treating one judge as ground truth.
Offline local JudgeProviderOptional CPU or CUDA llama.cpp judging with bounded, redacted inputs, strict JSON verdicts, evidence citations, and cached results.
Host-adapter latency baselinesDeterministic, redacted latency benchmarking per host event with immutable baseline records.

02 / CONTROL

Improve with gates

Promote artifacts only when controlled evidence supports a safe change.

Evidence-gated self-improvementImmutable prompt, tool, and skill candidates are evaluated offline, canaried, promoted, and rolled back through safety gates.
Safety-centered evolution controlsNo executable-code evolution; session-sticky assignment, fail-open telemetry and hooks, and promotion blocked on safety regressions.
Redacted replay cacheContent-addressed, immutable caching of evaluation stages for targeted reruns rather than full benchmark repeats.
Graph-lite temporal evidence retrievalISO date extraction and temporal routing for timeline queries without requiring a graph database.

03 / INTEGRATE

Preserve usable context

Carry evidence safely across harness and host boundaries.

Multi-evidence selectionSelects compact evidence unions for questions with multiple independent requirements and reports uncovered needs.
Structured-content preservationProtects tables, schedules, lists, and date-bearing facts from collapsing into unusable summaries.
EvidenceProvider dashboard contractVisible, capability-based integration boundaries for OpenClaw, Claude, Codex, and Hermes before retrieval paths activate.
Local integration packagingHost-neutral core plus generated OpenClaw, Claude, Codex, and Hermes integration artifacts.

The differentiator: Harness Ranger is an evidence-driven evaluation and controlled-evolution layer that can sit beside multiple agent harnesses, not merely another agent runtime.
